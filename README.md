# 🌙 Soulful Tarot Journey

A modern, minimal, and immersive spiritual website built with **Next.js** — designed to guide users through tarot, healing, and self-discovery.

---

## ✨ Overview

Soulful Tarot Journey is a beautifully crafted web experience offering:

* 🔮 Tarot Readings
* 🧿 Reiki Healing
* 🔢 Numerology Insights
* 🛍️ Spiritual Products

Built with performance, simplicity, and elegance in mind.

---

## 🚀 Tech Stack

* **Next.js**
* **React.js**
* **Tailwind CSS**
* **PostCSS**

---

## 📁 Project Structure

```
├── components/      # Reusable UI components
├── pages/           # Application routes
├── public/          # Images & assets
├── styles/          # Global styles
├── pages/api/       # Backend API routes
```

## ⚙️ Getting Started

### 1. Clone the repository

```
git clone https://github.com/abhyushdixit/soulfultarotjourney.git
cd soulfultarotjourney
```

### 2. Install dependencies

```
npm install
```

### 3. Run development server

```
npm run dev
```

Visit: `http://localhost:3000`

---

## 🔐 Environment Variables

Create a `.env.local` file in the root directory and add:

```
YOUR_ENV_VARIABLE=your_value
```

> ⚠️ Never commit `.env.local` to GitHub
